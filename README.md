# フランス社会文化芸術ゼミナール 公式サイト

このリポジトリは、フランスの社会・文化・芸術を研究するゼミナールの公式ウェブサイトです。

## サイトの目的

- ゼミの活動内容や研究テーマを発信する
- メンバー紹介や連絡先を掲載する
- フランスの社会文化芸術に関する情報を共有する

## 技術スタック

- HTML
- CSS
- JavaScript（必要に応じて）
